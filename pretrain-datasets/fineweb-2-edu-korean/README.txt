============================================================================
📚 The FineWeb2-Edu-Korean dataset is an educational web page filtered version
of the kor_Hang subset from the Fineweb2 dataset released by HuggingFaceFW.

USAGE
Depending on your purpose, select only data above a certain score and use it
for Korean language model pretraining.

LIMITATIONS / TODOS
- The filtering process requires significant computing power.
- The training quality has not been compared with the dataset before filtering.
- The classifier model has not been trained/evaluated.
